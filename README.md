# hciproject-jukenet
For HCI Project, Jukenet, an open source music player and playlist curator

All files (.png, .mp3) within this file were used for this project

To run, open the source code named index.html, and run it, it'll open up the site within your browser

Current Features
- Playlist Picker
- Music Picker
- Music Player

Non Functional Features [Possible future implementation]
- Search Feature
- Adding New Songs
- Adding More Playlists
